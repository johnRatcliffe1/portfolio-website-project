JavaScript Quiz



The JavaScript quiz was the second project I created. I used HTML, CSS and JavaScript to create a quiz that would ask the user 5 questions and then display their score at the end. I took the minimalist approach to the design of the quiz and used a simple colour scheme. I used JavaScript to create the quiz and used several functions to display the questions, answers and score at the end.
